HTML Elements Overview

List

HTML provides two types of lists: ordered lists with <ol> and unordered lists with <ul>. List items are marked with <li> tags.

Table

Tables are created using the <table> tag. Table rows are represented by <tr> and table data/cells by <td>. Table headers can be defined using <th>.

Meta Elements

Meta elements provide metadata about the HTML document. The <meta> tag can contain attributes like charset, name, and content to specify character sets, viewport settings, and other metadata.

Frames

Frames allow you to split the browser window into multiple sections, each displaying a separate HTML document. Frames are defined using the <frameset> tag and each frame is defined with the <frame> tag.
