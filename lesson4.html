HTML Forms and New Features

Forms

HTML forms are used to collect user input. They are defined using the <form> tag. Form controls like text fields, checkboxes, radio buttons, and buttons are added using various input types and tags like <input>, <textarea>, and <button>.

New Features

HTML5 introduced several new features and APIs to make web development easier and more powerful. Some of the notable features include:

  • Semantic Elements: Elements like <header>, <footer>, and <article> to better structure web pages.
  • Canvas: An HTML element used to draw graphics using JavaScript.
  • Video and Audio: Native support for embedding video and audio content using the <video> and <audio> tags.
  • Geolocation API: Allows web applications to access geographical location information of the user's device.
  • Web Storage: Provides mechanisms for storing data locally in the user's browser using localStorage and sessionStorage.
